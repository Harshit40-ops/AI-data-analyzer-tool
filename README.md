# 🚀 AI Data Analyzer – Intelligent Data Science Dashboard

An **AI-powered automated data analysis platform** built with **Python, Streamlit, and Machine Learning** that allows users to upload datasets and instantly generate **insights, visualizations, ML predictions, and professional reports**.

This project demonstrates **end-to-end data science workflow automation** — from **data cleaning → visualization → AI insights → machine learning → reporting**.

---

# 🌟 Key Features

### 📊 Smart Data Analysis

* Automatic dataset cleaning & preprocessing
* Instant dataset preview and statistics
* Missing value detection

### 📈 Advanced Visualization

* **100+ automatic charts**
* Interactive **Plotly visualizations**
* **3D data visualization**
* Custom **drag-and-drop graph builder**

### 🤖 AI & Machine Learning

* Automatic **multi-model training**
* Best model selection
* Feature importance analysis
* Model performance comparison

### 🧠 AI Insights Engine

* Automatically generates **data insights**
* Highlights correlations and patterns
* Helps users understand datasets faster

### 💬 Chat With Dataset

* Ask questions about the dataset
* Get instant answers from the data

### 📄 Automated Report Generation

* Generate **professional PDF reports**
* Includes dataset summary, statistics, insights, and ML results

### 📊 PowerBI-Style Dashboard

* Interactive sidebar controls
* Glass-UI styled analytics dashboard
* AI Auto Dashboard generator
